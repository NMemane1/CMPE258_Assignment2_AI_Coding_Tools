# Part D – Agentic MNIST

This section demonstrates an AI-assisted machine learning workflow using the MNIST handwritten digit dataset.

## Task
Train a neural network to classify handwritten digits.

## Dataset
MNIST (28x28 grayscale images of digits 0–9)

## Model
Simple neural network with:
- Flatten layer
- Dense hidden layer
- Softmax output layer

## AI Tool Used
Antigravity / AI coding assistant

## Result
The trained model achieves high classification accuracy on the MNIST test dataset.
